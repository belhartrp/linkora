# Linkora

**Linkora** adalah platform untuk membuat portofolio online secara instan. Pengguna tidak perlu menulis kode — cukup pilih template, isi data diri, sesuaikan tampilan, lalu hosting/publish dan portofolio langsung online.

## 🚀 Cara Menggunakan

### 1. Pilih Template
Pilih salah satu template portofolio yang sudah tersedia sesuai gaya dan kebutuhan (personal, profesional, kreatif, dll).

### 2. Isi Data
Masukkan data diri melalui form yang sudah disediakan, seperti:
- Nama & foto profil
- Bio/deskripsi singkat
- Daftar proyek/portofolio
- Pengalaman & skill
- Link sosial media

### 3. Custom Tampilan
Sesuaikan tampilan portofolio sesuai preferensi, seperti warna, font, dan layout, langsung dari editor tanpa perlu coding.

### 4. Hosting/Publish
Setelah selesai, klik publish untuk membuat portofolio langsung online dan bisa diakses lewat URL unik yang dapat dibagikan ke siapa saja.
